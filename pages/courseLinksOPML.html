MyOpenLibrary 2.0 - Course Feeds

Generate an OPML file of course related feeds so you can use Netvibes as a surrogate MyOpenLibrary environment.

For more infomation , see Course Profiles for MyOpenLibrary 2.0

Enter a course code and click the 'make courselinks.opml' button to get a list of rss feeds in OPML.

Possibly valid course codes include: A215, A217, A219, A251, A300, A430, A840, A841 A847, A850, A851, A857, AA307, AA307, AA311, AA312, AA315, AA820, AS208, E115, E123, E124, E131, E215, E243, E301, E848, ED840, H807, K113, K115, K202, K216, K221, K302, K311, KYP302, KZL107, M801, M886, S320, S390, S802, S804, S807, S809, S810, S819, SD805, SEH806, SXG390, SXM390, T307, T320, T324, T455, T883, T889, U211. However, note that support varies across course codes (the service creates links to all possible feeds, rather than ones that necessarily contain content; also, as courses cease to be presented, library support for them will disappear).

Note that not all OU courses have associated MyOpenLibrary resources. See MyOpenLibrary for more details.
