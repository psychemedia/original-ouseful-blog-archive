deliSearch Tools

Why not make the most of deliSearch using some of the following deliSearch integration tools?

del.icio.us integration

deliSearch bookmarklet (IE7 and Firefox): simply drag the bookmarklet onto your browser toolbar and use it to add deliSearch to any del.icio.us links page.

del.icio.us Greasemonkey script (Firefox with Greasemonkey extension): automatically add deliSearch to any del.icio.us links page.

widgets

Embeddable deliSearch widget: embed a deliSearch widget in your own blog. Configure the required delicious user name and tags from the widgetbox control panel. Download available from Widgetbox.

Tag aware embeddable deliSearch widget: embed a deliSearch widget in your own blog. Configure the required delicious user name and tags from the widgetbox control panel, and automatically detect appropriate tags from your blog page. Download available from Widgetbox.

user tips

Tag links you want to include in a searchroll with a particular tag (we're working on doing clever things with links tagged as usr:deliSearch).

If for some reason deliSearch is not pulling results into the results box, try this basic deliSearch interface.

deliSearch blog

There is no dedicated deliSearch blog at the current time. In the meantime, you can keep up to date by searching for deliSearch on the OUseful blog.

searchfeedr

searchfeedr is deliSearch generalised to feed off any* web feed (*in principle...).

Search for searchfeedr on the OUseful blog to find out more...

Stop press: this searchfeedr bookmarklet adds a searchfeedr search box to delicious user pages and Technorati tag pages (click on a tag then search the top feeds listed in it).

Note that the page search is not likely to produce many - if any - results when fed with a Technorati feed because the pages are so recent that they are unlikely to have been indexed by Yahoo!
