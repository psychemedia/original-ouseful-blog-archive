Simile Exhibit Library Catalogue Browser?

Just a quickie - I've just had a look at this post on Fun with SIMILE Exhibit, which takes a BibTex bibliography file and displays it as a Simile Exhibit.

Here's the result: a semantically browsable publications page. Go check it out...

Why's this interesting? Well, it strikes me that the Simile Exhibit code could be used to provide a client for the BibTex data (or the JSON formatted data that Simile actually feeds off)... so how about if a search on a library catalogue was to expose the results via BibTex, and a Simile Exhibit client was then used to provide the user interface/display?
