delicious2opml Converter

Tonight's hack - a delicious links to OPML converter - took an hour for some reason, not half an hour, and it's another of those things that I don't really have a use for just now (partly because there are alternatives that I've only just re-remembered - bah:-(, although someone may find it useful.

delicious2OPML generator

It reuses several searchfeedr routines that pull delicious content into the page via JSON, and then trashes through them to produce the OPML.

To use it, enter a delicious user name (if you don't specify one, popular is used instead) and an appropriate tag.

As you can see, the OPML is generated in a text box on the page. So in many ways it's not as useful as this delicious OPML link generator by Danny Ayers, which uses online XSLT processing to generate the OPML directly from a delicious feed.

That's the tool I'd completely forgotten about...
