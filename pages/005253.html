Even More OU Library RSS Feeds

Thanks to James McNulty's Midas period in the OU Library over the last week, in which everything he touched turned to RSS, there is now a host of new RSS feeds available over and above those mentioned here.

First off, a feed for User Loans info (you'll find this link doesn't work without some necessary authentication data - name and Library Barcode to be exact...).

Then a host of feeds related to MyOpenLibray. For example:

  • Ebooks: Example feed for course A300

  • Databases: Example feed for course A300

  • Journals: Example feed for course A300

  • ROUTES: Example feed for course A300

  • Reference Shelf: Example feed for course A300

  • OpenUniversity Links Example feed for course A300

  • Articles: Example feed for course A840

  • Help and Support: Example feed for course A840

    • NB To follow some of the links contained within the feeds you need to go in through the OU authentication system.

    To make consuming sets of feeds easier - for example on a per course basis - I've started playing around with OPML files. For example, try this OPML file of feeds for A300. It seems to work okay in Netvibes, at least...

    PS Another addition: an RSS feed of my overdue library books.
