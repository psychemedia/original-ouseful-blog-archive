OU Course Codes - A Web 2.OU Crown Jewel

In the days when this blog was dominated by library related concerns, I used to spend a lot of time working out how to use ISBNs as pivot points for various book related searches; (librarians, of course, don't rate ISBNs - they'd rather focus on the city a book was printed in...).

At the time, I produced a simple script to generate URLs that accepted an ISBN key - Amazon look ups, for example, or book cover images - you can find a link to the page from here: ISBN Playground. (The list may be a little out-of-date/increasingly incomplete by now...)

Somewhen around that time, (or perhaps even before, in the days before I blogged everything?) I also dabbled with a couple of webservices, in particular a course related book search (as well as an alternative course search):

This service looked up the set books associated with a course (in fact, it looked up the books students had to purchase for the course) and provided links for them through to the OU Library, as well as on Amazon and elsewhere, and also provided a couple of links to related course pages.

The service is no longer maintained, so many of the courses listed may have ceased presentation by now (and many of the newer current courses won't be listed), but it was an interesting exercise and one I;d like to revisit one day; not least because I'd like to put together a full RESTful API for it (and maybe tidy up the current SOAP API).

Anyway, anyway, as a precursor to that - possibly - I thought I'd start collecting URLs that are 'keyed' with an OU course code. This list is likely to grow over time (I'll try to maintain it), so if you're interested, it's probably worth bookmarking this page and revisiting it every so often...

  • Courses and qualifications pages: http://www3.open.ac.uk/courses/bin/p12.dll?C01CODE. For example: http://www3.open.ac.uk/courses/bin/p12.dll?C01T184; (also as a OUseful redirect, http://ouseful.open.ac.uk/T184);

  • Course reviews: http://www3.open.ac.uk/coursereviews/course.aspx?course=CODE. For example: http://www3.open.ac.uk/coursereviews/course.aspx?course=T184;

  • Course online registration page: https://css2.open.ac.uk/outis/1bd/o1bdHomePage.asp?CourseCode=CODE. For example: https://css2.open.ac.uk/outis/1bd/o1bdHomePage.asp?CourseCode=T184;

  • Course description PDF: http://www3.open.ac.uk/courses/pdfs/CODE.pdf. For example: http://www3.open.ac.uk/courses/pdfs/T184.pdf;

  • ROUTES recommended web links (lite results): http://routes.open.ac.uk/ixbin/hixclient.exe?_IXDB_=routes&_IXSPFX_=h&_IXNOMATCHES_=nomatch1.html&_IXMAXHITS_=50&submit-button=summary&%24rec+resource=.&%24with+resource_visible=.&%24sort+title=.&%24%3Fquery%3A%28%24query%29+in+%28title+description+keywords+courses+alternative_title%29=.&%24%3Dquery=CODE.   • Excluded combinations: http://www3.open.ac.uk/courses/bin/p12.dll?C07CODE. For example: http://www3.open.ac.uk/courses/bin/p12.dll?C07T209;(also as a OUseful redirect, http://ouseful.open.ac.uk/T209x);

  • Course code as a keyword search item in the OU Library Catalogue: http://voyager.open.ac.uk/cgi-bin/Pwebrecon.cgi?Search_Arg=CODE&Search_Code=FT*&CNT=22. For example: http://voyager.open.ac.uk/cgi-bin/Pwebrecon.cgi?Search_Arg=t209&Search_Code=FT*&CNT=22;
    (also as a OUseful redirect, http://ouseful.open.ac.uk/library/t209);

  • OpenLearn Units by OU Course Code (RSS feed via Yahoo Pipes): http://pipes.yahoo.com/pipes/pipe.run?_id=eBdDcNdm3BGw5cCHy6ky6g&_render=rss&ccode=CODE. For example: http://pipes.yahoo.com/pipes/pipe.run?_id=eBdDcNdm3BGw5cCHy6ky6g&_render=rss&ccode=T180; (pipe 'homepage');

  • OU Course code search on Open2.net (RSS feed via Yahoo Pipes): http://pipes.yahoo.com/pipes/pipe.run?_id=EKK4eOFm3BGYFnuey6ky6g&_render=rss&ccode=CODE. For example: http://pipes.yahoo.com/pipes/pipe.run?_id=EKK4eOFm3BGYFnuey6ky6g&_render=rss&ccode=T175; (pipe 'homepage');

  • Relevant Knowledge short courses (requires authentication): http://students.open.ac.uk/technology/courses/CODE. For example: http://students.open.ac.uk/technology/courses/tu120;

  • OU VLE courses (requires authentication): http://learn.open.ac.uk/CODE-PRESENTATION. For example: http://learn.open.ac.uk/t175-07b;

  • Second hand course materials: http://www.universitybooksearch.co.uk/search.asp?startsearch=TRUE&Code=CODE. For example: http://www.universitybooksearch.co.uk/search.asp?startsearch=TRUE&Code=T171;

Here's a host of RSS feeds sourced from MyOpenLibray. (As the library is about to undergo a redesign, these URLs may not be very stable, if indeed they work at all... assume it's broken if it's not listed as (works)!):

  • Ebooks: http://myopenlibrary.open.ac.uk/rss/rss_ebooks.php?course=CODE; for example, Example feed for course A300 (works);
  • Databases: http://myopenlibrary.open.ac.uk/rss/rss_databases.php?course=CODE; for example, Example feed for course A300 (works)
  • Journals: http://myopenlibrary.open.ac.uk/rss/rss_ejournals.php?course=CODE; for example, Example feed for course T324 (works);
  • ROUTES: http://myopenlibrary.open.ac.uk/rss/rss_routes.php?course=CODE; for example, Example feed for course A300
  • Reference Shelf: http://myopenlibrary.open.ac.uk/rss/rss_reference.php?course=CODE; for example, Example feed for course A300
  • OpenUniversity Links: http://myopenlibrary.open.ac.uk/rss/rss_links.php?course=CODE; for example, Example feed for course T324 (works)
  • Articles: http://myopenlibrary.open.ac.uk/rss/rss_recread.php?course=CODE; for example, Example feed for course A840 (works);
  • Help and Support: http://myopenlibrary.open.ac.uk/rss/rss_librarians.php?course=CODE; for example, Example feed for course A840 (works);

The above library feeds can also be bundled using this library course feed bundle OPML generator - which i will update at some point to run as a live OPML feed generator keyed by a course code.

It's worth noting, however, that the library database is sparsely populated and not all courses have items populating the above feeds.
