Course Links OPML Generator

In a couple of previous posts (here and here), I have mentioned how Netvibes can be used as an alternative interface to MyOpenLibrary by importing MyOpenLibrary RSS feeds.


I have just posted a draft version of a page here that wil create an OPML file when provided with an OU course code.

This file can then be uploaded into Netvibes to provide a set of feeds appropriate to that course.
