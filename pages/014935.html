Google Maps Catchup Post

Earlier this year I did quite a few posts relating to Google Maps, (Google Calendar maps, geoblogging (blogging from Google MyMaps), a geoTwitter map, some maps relating to school catchment areas, and so on) but didn't post everything...

So when an email came round last week asking whether anyone knew how to get postcodes plotted onto maps, I sent a reply:

Q: I need to create a simple map - plot some sites onto a map of the UK using postcodes (or town names) and then code the markers to denote different site characteristics.

A: One way would be to put the details into a spreadsheet, pass the result through a geocoder (to convert postcodes to lat/long) and then plot different markers on a map.

Here's a recipe for getting the markers onto a map from a spreadsheet with a postcode column: http://www.andydickinson.net/2008/02/29/maps-spreadsheets-yahoo-pipes-and-post-offices/

I produced a couple of maps that will do client side processing relating to markers before plotting them: ./013580.html

Looking back at the second of the above mentioned posts, (and spotting the pipe had rotted - now fixed...), I realised that I hadn't posted anywhere the final extent of my tinkerings of the time. And looking around on my laptop, the code appeared to have gone :-( Anyway, fortunately I'd posted a nearly-as-far-as-I'd-got edition of the code to the OUseful blog server, so before I lose it again, here's a link: Generic Circular Catchment Areas - Interactive Map

What the map allows you to do is load in a set of points from a Google MyMap, marked up as members of different classes, and then plot the catchment areas around each point using a customisable radius for each class. (Along with a user defined radius, you can also set the colour of each circular catchment area (using a YUI colour selector)).

You can hide all the markers from any class, or just hide individual points.

(The URL on the map page for the Plymouth supermarkets MyMap is incorrect. Here's the correct link: Plynmouth supermarkets MyMap.)

When I get a chance, I'll try to tidy up the code and do a version that: a) will plot points identified by postcode from a Google spreadsheet; b) let oyou use different markers as well as or instead of the circular catchment areas; c) have a think about how a table containing postcode info could be progressively enhanced with a map display.
