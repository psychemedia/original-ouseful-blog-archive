Scribd iPaper Bookmarklet Generator

This web page application (only tested in Firefox) will create a bookmarklet that you can add to your browser and that will convert any links to documents linked to from a web page to links that allow you to view the document using Scribd's iPaper, or that let you replace those links with actual embedded versions of the documents using iPaper.

To use the bookmarklet, you'll need a Scribd develop key, becuase the documents you view have to be uploaded to a Scribd account. You can get the developer key by registering with the Scribd Platform.
