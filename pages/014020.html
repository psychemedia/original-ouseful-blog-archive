All I want is something like this:

DOI: (in the form, e.g. doi:10.1002/0470841559.ch1

(though nicely styled, of course, and checking for a leading doi: prefix, adding one if it's not there...)

Here's the core of it, in code form... (the above form is a real form, not a screenshot, so it should work...)

<form name="doi resolver" action="http://openurl.open.ac.uk.libezproxy.open.ac.uk/sfxlcl3"

DOI:
<input type="text" name="id" value="doi:10.1002/0470841559.ch1" size="50"> (in the form, e.g. <em>doi:10.1002/0470841559.ch1</em>

(It uses OU credentials to provide me with authenticated access to the resource, so the above is not much use to you if you don't have an OUCU...)
