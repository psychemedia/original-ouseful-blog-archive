gCalMap - Embed Code Generator For Google Calendar Maps

A really quick post, this one, just to say I've posted a gCalmap embed code generator at http://ouseful.open.ac.uk/maps/gCalMapGenerator.php:



Enter the a public Google Calendar ID and generate URL for just the calendar map, or a side-by-side calendar and map view.

I've also added a feature that will generate an iframe embed code, to make it a little easier to embed these views in your own pages.

When I get a chance, I'll add the multi-calendar view to the page, too...

PS the mashup has gone wild... (though not using the generator...): http://www.martinwunderlich.com/foss-calendar.htm
