Grazring Grazr OPML Uploads: Bookmarklet

It's late, I'm in Belfast (taking part in a two day brainstorming session planning the W5 Robot Thought show (which may make use of the Robot Amusement Park activity we've been developing elsewhere as one of the schools' launch day activities) and as I probably won't be posting for a day or two, here's a little something to be playing with...

A bookmarklet that opens an OPML file uploaded to Grazr in a Grazr navigation window: Grazr OPML upload navigator bookmarklet.

That is, if you visit the 'homepage' of an OPML file that has been uploaded to Grazr (such as this one) and click on the bookmarket, it will open the OPML file in a 'navigation sidebar' Grazr widget; clicking on any HTML link within the widget will then open the corresponding page in a neighbouring iframe within the same browser window.
