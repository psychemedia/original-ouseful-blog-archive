Yahoo Pipes Badges Make It Easy to Share Google Calendar Maps

A comment on the previous post pointed to a use case example of how to embed a Google Calendar Map (gCalmap) in a blog entry using a Yahoo Pipes map badge: Mapa de conciertos de R.E.M. 2008 (Map of REM 2008 Tour dates (if anyone from the REM production team is reading, three passes to the Southampton gig would be much appreciated ;-):

The badges also have a link that makes it easy to share the map (see the Get This >> link bottom right?)

An element of badge customisation is possible too: for example, I can set the size and change the badge type (the gCal XML is the user input specified in the original pipe):

That said, it would be really handy to be able to change the badge title, particularly for utility pipes like the gCalMap pipe, where any given map will correspond to a particular calendar/series of events (and will thus merit its own title; in this case "REM Tour Dates, 2008", for example).

The embed code is actually a javascript include, which means it probably won't show through in a feed reader, though you should be able to see it if you click through:

(If you need to embed an iframe, then there's still muy own gCalMap embed code generator.)

It could be handy to be able to see the pipe's info/description information in the pop-up panel, as well as a few more customisation options, but maybe that's on the cards already?
