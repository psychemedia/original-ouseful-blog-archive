GeoRSS from delicious Geo-tags via a Yahoo pipe

Although there are now several ways to create geo-tagged feeds using map based interfaces - for example using Google Mymaps - there may still be occasions when you want to get geo-tagged resources from social bookmarking systems such as delicious (err, maybe...?! ;-)

So here's a Yahoo Pipe that will do just that. It looks for tags of the form geo:lat= and geo:long= in a delicious feed and if it finds them it augments the feed item with corresponding <geo:lat> and <geo:long> tags:

Here's the pipe: delicious geoRSS annotation pipe
