Multisearch panels demo

The aim of this page is just to demonstrate various ways of getting several search engines displayed together, to encourage students to try out different search engines.

Note that the approach used in this demo will incur quite large downloads - each frame is loaded with a search engine homepage. It would be possible to optimise the page and only download a search engine homepage when the tab is opened for the first time.
