Scriptaculous Slider Code Generator

As I've spent several hours on trains over the last couple of days, I thought I spend the time in idle code recreation, and the result is a slightly ropey, almost functional (in Firefox at least) code generator for a script.aculo.us slider.

Full functionality is not there yet - it doesn't even have an option to use an image as a slider yet - but I have run out of time allowed on this curiosity, so unless I get more than a couple of requests for this feature to be added, it won't...

Here's a screenshot (if anyone would like to give me a style file, that'd be great;-)

In case you don't know it, script.aculo.us is a javascript library that provides all sorts of neat animation effects that can an easily incorporate in your own pages.
